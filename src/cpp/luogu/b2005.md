# 字符三角形

## 题目描述

给定一个字符，用它构造一个底边长 $5$ 个字符，高 $3$ 个字符的等腰字符三角形。

## 输入格式

输入只有一行，包含一个字符。

## 输出格式

该字符构成的等腰三角形，底边长 $5$ 个字符，高 $3$ 个字符。

## 样例 #1

### 样例输入 #1

```
*
```

### 样例输出 #1

```
*
 ***
*****
```

## 提示

对于 $100 \%$ 的数据，输入的字符是 ASCII 中的可见字符。

## 题解

```cpp
#include <iostream>
using namespace std;
int main(){
    char a;
    cin >> a;
    cout <<"  "<<a<<endl<<" "<<a<<a<<a<<endl<<a<<a<<a<<a<<a;
    return 0;
}
```
